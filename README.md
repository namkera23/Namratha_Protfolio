## 📦 Installation & Setup

```bash
# Clone the repo
git clone https://github.com/pradnyajagtap-3660/Pradnya_Protfolio.git

# Install dependencies
npm install

# Run Tailwind build process
npm run build

# Start the server (if using Express backend)
npm start
